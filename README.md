# Shan's Cookbook

Shan's Cookbook is a personal recipe website I built for my fiancée to keep her recipes organised in one place and make them easier to browse.

Recipes are stored in JavaScript and displayed across different categories, including favourites, must-make recipes and easy meals.

## Features

* Recipe cards generated from JavaScript data
* Separate recipe categories
* Horizontally scrollable recipe rows
* Responsive layout for desktop and mobile
* Custom pastel design
* Pom Easter egg with sound and confetti
* Interactive warning after repeated clicks

## Built With

* HTML
* CSS
* JavaScript

## Project Status

The main cookbook interface is working and I plan to continue expanding the website as more recipes are added.

Individual recipe pages and further improvements will be added as the project develops.

## Running Locally

Clone the repository and open `index.html` in your browser.

For development, you can also run the project using the Live Server extension in VS Code.
